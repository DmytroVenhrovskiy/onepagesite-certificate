10 Базових
Команд GIT:

  • 1 - git init - < Створення нового локального репозиторію в поточному каталозі >
  • 2 - git clone <> - < Клонування існуючого віддаленого репозиторію на локальний комп'ютер >
  • 3 - git add <> - < Додає зміни у файлі до індексу для подальшого коміту >
  • 4 - git commit -m - < Створює новий коміт з усіма змінами, доданими до індексу >
  • 5 - git status - < Показує стан робочої директорії, які файли змінені, готові до коміту або не відстежуються >
  • 6 - git log - < Показує повну історію комітів >
  • 7 - git branch - < Переглядає всі існуючі гілки >
  • 8 - git branch <ім'я_гілки> - < Створює нову гілку >
  • 9 - git checkout <ім'я_гілки> - < Перемикається на іншу гілку >
  • 10 - git push <віддалений_репозиторій> <ім'я_гілки> - < Відправляє локальні зміни на віддалений репозиторій >
